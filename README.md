# 📊 Project Overview

This project is an end-to-end **Data Analyst Portfolio Project** built using **MySQL** and **Power BI**.

The project starts with collecting the Olympics dataset from **Kaggle**, followed by cleaning and preparing the data using **MySQL**. After transforming the dataset into a clean and analysis-ready format, an interactive dashboard was created in **Power BI** to visualize key insights about Olympic medals, athletes, sports, cities, and countries.
